Makefile

可以通过make install将hello直接安装到/usr/bin目录。也可以通过make install DESTDIR=/tmp/test将hello安装在/tmp/test/usr/bin目录。

稍微复杂一点的需要定义PREFIX，一般autotools工程，会运行这样的指令:

./configure -prefix=/usr或者./configure --prefix=/usr/local来指定PREFIX

Makefile改写为

HelloWorld安装方法：需要引入变量CMAKE_INSTALL_PREFIX，该变量类似configure脚本的-prefix，

cmake -DCMAKE_INSTALL_PREFIX=/usr

INSTALL指令用于定义安装规则，安装的内容可以包括目标二进制、动态库、静态库以及文件、目录、脚本等。
